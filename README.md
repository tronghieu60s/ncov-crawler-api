# ncov-crawler-api

API Data Viruss Covid-2019 (VietNam and Global).
https://www.worldometers.info/coronavirus/


  - Express
  - Node-crawler
  - Mongoose

### Installation

ncov-crawler-api requires [Node.js](https://nodejs.org/) v4+ to run.

Install the dependencies and run start server.

```sh
$ cd ncov-crawler-api
$ npm install
$ npm start
```
